# AWS Static Website Hosting 

Hosted a personal portfolio website using **Amazon S3 Static Website Hosting**.

##  Services Used
- Amazon S3
- IAM (Bucket Policy)

##  Features
- Public website hosting via S3
- Custom HTML & CSS portfolio
- Responsive design
